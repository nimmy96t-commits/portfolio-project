# Full Stack Portfolio with CI/CD Pipeline

## 📌 Project Overvieww

This is a full-stack portfolio web application that showcases projects and allows users to submit messages through a contact form.
It demonstrates frontend, backend, database integration, and CI/CD pipeline.

---

## 🚀 Tech Stack

* Frontend: HTML, CSS
* Backend: Node.js, Express
* Database: PostgreSQL (Render)
* Version Control: Git & GitHub
* Deployment: Render
* CI/CD: GitHub Actions + Render

---

## ✨ Features

* Responsive portfolio website
* Contact form
* Backend API integration
* Data stored in PostgreSQL
* Hosted on cloud
* Automated CI/CD pipeline

---

## 🌐 Live Website

https://portfolio-project-bsud.onrender.com

---

## ⚙️ CI/CD Pipeline

This project uses a complete CI/CD pipeline:

* **CI (Continuous Integration):**

  * Triggered on every git push
  * GitHub Actions installs dependencies and runs server test

* **CD (Continuous Deployment):**

  * Render automatically deploys the latest code

### 🔄 Workflow

Push Code → GitHub Actions (Test) → Render Deploy → Live Website

# 🌐 Full Stack Portfolio with CI/CD Pipeline

## 📌 Project Overview

This is a full-stack portfolio web application that showcases personal details and allows users to send messages through a contact form.  

The project demonstrates frontend development, backend API integration, and deployment using a CI/CD pipeline.

It follows a basic client-server architecture where the frontend communicates with the backend using REST APIs.

---

## 🚀 Tech Stack

- **Frontend:** HTML, CSS  
- **Backend:** Node.js, Express  
- **Database:** In-memory storage (can be extended to PostgreSQL)  
- **Version Control:** Git & GitHub  
- **Deployment:** Render  
- **CI/CD:** GitHub Actions + Render  

---

## ✨ Features

- Responsive portfolio website  
- Contact form for user interaction  
- Backend API to handle form submissions  
- Messages stored temporarily on server  
- Admin messages page to view submitted data  
- Hosted on cloud (Render)  
- Automated CI/CD pipeline

## ⚙️ CI/CD Pipeline

This project uses a basic CI/CD pipeline:

### 🔹 CI (Continuous Integration)
- Triggered on every Git push  
- GitHub Actions installs dependencies  

### 🔹 CD (Continuous Deployment)
- Render automatically deploys the latest code  

---

## 🔄 Workflow

Push Code → GitHub Actions → Render Deploy → Live Website

---

## 📊 Project Behavior

- Users can submit messages via contact form  
- Messages are stored temporarily in server memory  
- Messages can be viewed in admin page (`/messages.html`)
